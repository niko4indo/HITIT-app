# 极简 PWA 记账系统 下面是完整可运行版本。 部署后： * 手机可安装成 APP * 自动本地存储 * 极简输入 * 支持 Rp / ¥ * 年底目标余额 * 日均可花 * PWA 离线运行 --- # 1. index.html ```html 极简记账

极简记账

当前余额

Rp 0

Rp 0

``` --- # 2. manifest.json ```json { "name": "极简记账", "short_name": "记账", "start_url": "./index.html", "display": "standalone", "background_color": "#ffffff", "theme_color": "#ffffff", "icons": [ { "src": "icon.png", "sizes": "192x192", "type": "image/png" } ] } ``` --- # 3. sw.js ```javascript const CACHE = 'budget-v1'; self.addEventListener('install', e => { e.waitUntil( caches.open(CACHE).then(cache => { return cache.addAll([ './', './index.html', './manifest.json' ]); }) ); }); self.addEventListener('fetch', e => { e.respondWith( caches.match(e.request).then(res => { return res || fetch(e.request); }) ); }); ``` --- # 4. icon.png 准备一个： * 192×192 * 白底图标 命名： ```text icon.png ``` --- # 部署方法 推荐： * GitHub Pages * Netlify * Vercel 上传以上文件即可。 --- # 手机安装 安卓 Chrome： ```text 菜单 → 添加到主屏幕 ``` 之后： * 像 APP 一样打开 * 本地自动保存 * 不会出现书签失效 * 可离线使用
